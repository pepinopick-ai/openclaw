## Режим 2 — Диагностика симптома → климат

| Симптом                        | Климатическая причина   | Проверить          |
| ------------------------------ | ----------------------- | ------------------ |
| Конденсат на листьях утром     | T° ночь ниже точки росы | RH + T° ночью      |
| Белый налёт                    | RH >85% без вентиляции  | Вентиляция + RH    |
| Увядание при нормальном поливе | T° >30°C                | Притенение + вент  |
| Медленный рост                 | CO2 <600 или PPFD <200  | CO2 + PPFD         |
| Деформированные плоды          | T° ночь <16°C           | Отопление          |
| Опадение завязей               | T° ночь >22°C           | Ночная вентиляция  |
| Остановка роста грибов         | T° >20°C или RH <80%    | Охлаждение + туман |

## Режим 3 — Расчёт точки росы

Формула Magnus:

```
Td = (243.04 × (ln(RH/100) + 17.625×T/(243.04+T))) /
     (17.625 - (ln(RH/100) + 17.625×T/(243.04+T)))
```

Конденсат возникает когда T° листа ≤ Td.

Профилактика:

- Снижать RH до <80% за 1–2 ч до темноты
- T° воздуха > T° точки росы + 3°C (запас)
- Циркуляционные вентиляторы — минимальный ночной обдув

## Режим 4 — Матрица коррекции

| Проблема      | Быстрое действие      | Системное решение       |
| ------------- | --------------------- | ----------------------- |
| T° > max      | Открыть фрамуги, вент | Затенение 30–50%        |
| T° < min      | Включить отопление    | Проверить теплоизоляцию |
| RH > max      | Усилить вентиляцию    | Разредить посадки       |
| RH < min      | Туман / дождевание    | Мульча                  |
| CO2 < 600 ppm | CO2 генератор         | Проверить герметичность |
| PPFD < норма  | Досвечивание          | Чистка/замена ламп      |
| Конденсат     | Циркуляция + вент     | Ревизия RH-режима ночью |
