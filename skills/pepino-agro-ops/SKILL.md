# Pepino Agro-Ops — Ежедневные агрономические операции

Управляет тремя зонами теплицы и ведёт операционный журнал.

**Зоны:**

- **Zone-A** — Огурцы (гидропоника)
- **Zone-B** — Грибы (вешенка + шиитаке)
- **Zone-C** — Микрозелень + Цицак

## Режим 2 — Ранняя диагностика болезней

| Симптом             | Вероятная причина     | Приоритет | Действие                            |
| ------------------- | --------------------- | --------- | ----------------------------------- |
| Жёлтые листья (низ) | Дефицит N / старение  | WARN      | Проверить EC, добавить N            |
| Коричневые пятна    | Botrytis / Alternaria | CRIT      | Снизить влажность, фунгицид         |
| Белый налёт         | Мучнистая роса        | CRIT      | Вентиляция, серный спрей            |
| Увядание днём       | Недополив / Fusarium  | CRIT      | Проверить полив + субстрат          |
| Плесень на блоках   | Конкурентный гриб     | CRIT      | Изолировать, проверить стерильность |
| Жёлтые кольца       | Вирусная инфекция     | CRIT      | Карантин, удалить растение          |

**Правило эскалации:**

- 1 симптом → наблюдать, записывать
- 2+ симптома или 20%+ поражения → АЛЕРТ в Sheets
- Распространение 2+ зон → КРИТИЧЕСКИЙ АЛЕРТ → `pepino-qa-food-safety` + `pepino-risk`

## Режим 7 — Post-Harvest (послеуборочная обработка)

**Источник:** postharvest (MASTER_INDEX merge)

**Триггеры:** "послеуборочная", "postharvest", "грейдинг", "сортировка", "холодная цепь", "cold chain", "упаковка", "shelf life", "срок годности", "хранение", "температура хранения"

**Справка SOP:** SOP-QA-003 (Послеуборочная обработка и контроль качества)

### Grading Standards

| Продукт     | Grade A                          | Grade B                     | Grade C (переработка)         |
| ----------- | -------------------------------- | --------------------------- | ----------------------------- |
| Огурцы      | 15-22 см, ровные, без пятен      | Лёгкая кривизна, мелкие     | Сильная кривизна, повреждения |
| Вешенка     | Шляпка 3-8 см, плотная, целая    | Мелкие или слегка подсохшие | Повреждённые, переросшие      |
| Шиитаке     | Шляпка 4-8 см, трещины на шляпке | Мелкие, без трещин          | Повреждённые, размякшие       |
| Микрозелень | Равномерный рост, яркий цвет     | Неравномерный, бледноватый  | Пожелтевший, переросший       |
| Цицак       | 8-15 см, зелёный/жёлтый, упругий | Мелкие, лёгкие повреждения  | Мягкие, пятна                 |

**Правила:**

- Grade A → прямые продажи ресторанам (полная цена)
- Grade B → продажа со скидкой 15-20% или переработка (ферментация)
- Grade C → только переработка (pepino-fermentation) или компост
- Доля Grade A целевая: > 80% от урожая
