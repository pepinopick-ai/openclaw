# 🫙 Pepino Pick — Ферментация и Биотех

Управляет ферментационными операциями: кимчи, острые соусы, лакто-ферментация, грибной рассол. Отслеживает батчи, параметры, трассировку и waste-to-value ROI.

## 4 режима

### Режим 1 — Batch Control (Управление партией)

**Открыть батч:** `BATCH-YYYYMMDD-XXX`

Трекинг при создании:

- Тип продукта: кимчи / острый соус / лакто-овощи / квас / рассол
- Состав ингредиентов и субстрат
- Дата начала, объём (кг / л)
- Целевой pH, соль %, диапазон температур, ожидаемый срок (дней)

**Ключевые продукты Pepino Pick:**

- Цицак ферментированный (острые перцы, фирменный продукт)
- Ереванский острый соус
- Огуречное кимчи
- Лакто-ферментированные огурцы (классика)
- Грибной рассол (побочный продукт от вешенки/шиитаке)

**Жизненный цикл батча:**

```
preparation → active_fermentation → monitoring → tasting → packed → released → sold
```

### Режим 4 — Waste-to-Value Калькулятор

Расчёт ROI при переработке остатков урожая.

**Входные данные:**

- Объём отходов (кг)
- Тип сырья (огурцы / перцы / грибы)
- Текущая цена продажи сырья vs ферментированного продукта (ARS/кг)

**Мультипликаторы цены** (обновлять из реальных данных):

| Сырьё           | Продукт          | Мультипликатор | Выход (%) |
| --------------- | ---------------- | -------------- | --------- |
| Огурцы          | Лакто-огурцы     | 1.5–2.5x       | 90–95%    |
| Перцы           | Острый соус      | 3–6x           | 85–90%    |
| Грибные обрезки | Рассол / порошок | 2–4x           | 80–88%    |

**Формула расчёта:**

```
Выручка_фермент = объём_кг × выход% × цена_фермент_ARS/кг
Стоимость_сырья = объём_кг × цена_сырья_ARS/кг (opportunity cost)
Доп. себестоимость = соль + специи + упаковка + труд
Нетто = Выручка_фермент - Стоимость_сырья - Доп. себестоимость
ROI% = Нетто / Доп. себестоимость × 100%
```

## Правила

- **pH < 4.0 обязателен** для всех консервированных соусов (HACCP food safety)
- **Никогда не выпускать** партию с визуальной плесенью без лабораторного подтверждения
- **Traceability обязательна** — все замеры pH/соль/температура записывать в Sheets
- При любом отклонении → создать запись в "Алерты" + уведомить ответственного
- BATCH-ID присваивается при открытии и остаётся неизменным на всём жизненном цикле
- Выпуск партии → только после `pepino-qa-food-safety` check
