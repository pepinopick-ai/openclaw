## Режимы работы

### 1. РЕЕСТР ДОГОВОРОВ

**Типы договоров:**

- Клиентские (рестораны, оптовики)
- Поставщики (субстрат, мицелий, упаковка)
- Трудовые (сотрудники, подрядчики)
- Аренда (помещение теплицы, земля)
- Партнёрские (коллаборации, дистрибуторы)
- NDA (конфиденциальность)

**Карточка договора:**

```
ID: DOC-[YYYYMMDD]-[тип]-[контрагент]
Тип: [клиентский / поставщик / трудовой / аренда / партнёрский / NDA]
Контрагент: [название / ФИО]
Дата начала: [дата]
Дата окончания: [дата] / Бессрочный
Сумма: [ARS/мес или общая]
Ключевые условия: [краткое описание]
Статус: [active / expired / terminated / draft]
Файл: [Google Drive ссылка]
Требует продления: [да/нет, за N дней]
Linked case: [CASE-YYYYMMDD-LEG]
```

**Алерты:**

- 🟡 30 дней до истечения договора
- 🔴 7 дней до истечения договора
- 🔴 Договор истёк

### 2. РЕГУЛЯТОРИКА И LICENSES (АРГЕНТИНА)

**Ключевые регуляторные требования:**

**SENASA** (Servicio Nacional de Sanidad y Calidad Agroalimentaria):

- Регистрация производителя свежих грибов и овощей
- Habilitación для работы с пищевыми продуктами
- Требования к хранению и транспортировке (холодовая цепь)
- Labeling requirements для упакованной продукции

**Bromatología municipal:**

- Habilitación торгового заведения / производства
- Санитарные требования к помещению
- Периодические инспекции

**Fermentados:**

- Registro de producto ante ANMAT (для промышленной реализации)
- pH контроль (ниже 4.6 для безопасности)
- Labeling: ингредиенты, аллергены, дата изготовления, срок годности

**Трудовое право:**

- Регистрация сотрудников в AFIP
- Modalidades de contratación (relación de dependencia vs monotributista)
- Aportes y contribuciones patronales

**Чеклист "Готовность к реализации" (новый SKU):**

```
☐ Рецептура задокументирована
☐ Lote/batch система работает
☐ Этикетка содержит: producto, marca, peso neto, ingredientes, alergenos,
  fecha elaboración, vencimiento, RNPA/RNPPA (если нужно), datos del elaborador
☐ Условия хранения указаны
☐ pH задокументирован (для ферментов)
☐ Habilitación производителя актуальна
☐ Договор с клиентом / накладная
```

### 3. ШАБЛОНЫ ДОГОВОРОВ

**Клиентский договор (ресторан/оптовик):**
Ключевые пункты для включения:

```
- Ассортимент и минимальный заказ
- Цены и порядок изменения (привязка к инфляции или пересмотр каждые N мес)
- Сроки оставки и доставки
- Условия оплаты (предоплата / отсрочка N дней)
- Качество продукции и рекламации
- Порядок расторжения (N дней уведомления)
- Конфиденциальность
- Форс-мажор
```

**Договор с поставщиком:**

```
- Спецификации на поставляемый товар (качество, объём, ГОСТ/стандарт)
- Цена и условия пересмотра
- Сроки поставки, штрафы за задержку
- Условия возврата некачественного товара
- Гарантии поставщика
```

**NDA (соглашение о конфиденциальности):**

```
- Что является конфиденциальной информацией (рецептуры, клиентская база, финансы)
- Срок действия (2-5 лет)
- Исключения (публично известная информация)
- Санкции за нарушение
```

### 4. ПРЕТЕНЗИИ И СПОРЫ

**Процедура при претензии от клиента:**

```
1. Зафиксировать претензию (дата, суть, клиент, партия продукции)
2. Открыть case в pepino-qa-food-safety для трассировки batch
3. Оценить обоснованность: есть ли основания по договору?
4. Ответ клиенту в течение 48 часов
5. Если обоснована → компенсация (замена товара / скидка / возврат)
6. Если необоснована → аргументированный отказ
7. Зафиксировать в реестре для статистики
```

**Реестр претензий:**

```
[Дата] — [Клиент] — [Суть] — [Batch] — [Решение] — [Статус]
```

**Эскалация:**

- До 50,000 ARS → решает операционный менеджер
- До 500,000 ARS → решает CEO
- Выше или судебная угроза → внешний юрист

### 5. НАЛОГИ И ФИНАНСОВАЯ РЕГУЛЯТОРИКА

**Базовый контекст (Аргентина 2026):**

- Monotributo: лимиты категорий, даты переоценки (enero/julio)
- Responsable Inscripto: IVA 21%, percepción, retención
- AFIP: factura electrónica (tipo A/B/C), vencimientos mensuales
- Ingresos Brutos: ставки по провинциям, минимальные не облагаемые

**Годовой налоговый календарь:**

```
Enero: revisión categoría monotributo
Marzo: vencimiento ganancias personas físicas (según terminación)
Julio: revisión categoría monotributo
Mensual: vencimiento IVA (si Resp. Insc.), IIBB, pago monotributo
```

**Алерты по налогам:**

- 🟡 7 дней до вencimiento
- 🔴 1 день до vencimiento
