# 📊 Pepino Pick — Финансовый Контроллер

Мониторинг бюджет vs факт, маржинальный контроль, unit economics, cash flow. Генерирует алерты при отклонениях и поставляет данные для CEO briefing.

**Google Sheets ID:** `1AB9nkHfCu8_12dwn72tWKZOiqHAfAhI3AHOjeAAByoc`

- **Финансы** — фактические транзакции (приход/расход)
- **Стратегия** — плановые показатели / бюджет

---

## 4 модуля

### Модуль 1 — Budget vs Actual (Бюджет vs Факт)

Читает план из "Стратегия", факт из "Финансы". Считает отклонения за период (неделя / месяц / квартал).

**Метрики:**

- Выручка: план vs факт, δ ARS, δ %
- COGS (себестоимость): план vs факт
- Валовая маржа: план vs факт, δ процентных пунктов
- Операционные расходы по категориям
- Нетто прибыль: план vs факт

**Пороги алертов:**

| Показатель             | 🟡 WARNING | 🔴 CRITICAL    |
| ---------------------- | ---------- | -------------- |
| Выручка ниже плана     | > -10%     | > -20%         |
| Маржа ниже плана       | > -5pp     | > -10pp        |
| Расходы превышают план | > +15%     | > +30%         |
| Нетто отрицательный    | любое      | < -500,000 ARS |

### Модуль 2 — Margin Control (Маржинальный контроль)

Мониторинг маржи по продуктам / SKU в реальном времени (с учётом ARS инфляции).

**Продукты для трекинга:**

- Огурцы (свежие)
- Вешенка
- Шиитаке
- Цицак / Ереванский соус (ферментированные)
- Лакто-ферментированные огурцы
- Микрозелень

**Расчёт для каждого SKU:**

- Gross margin (выручка − COGS) / выручка × 100%
- Contribution margin (маржинальный вклад)
- Break-even volume (кг/мес)
- Тренд маржи: нед-к-нед, мес-к-мес

**Алерт:** маржа упала > 5pp vs прошлый месяц → авто-рекомендация пересмотра цен через `pepino-argentina-finance`

### Модуль 3 — Unit Economics Dashboard

**По каждой продуктовой категории:**

- Выручка ARS/кг
- COGS ARS/кг (субстрат + труд + энергия + упаковка)
- Маржа ARS/кг
- Маржа ARS/м² теплицы в неделю ← **ключевой KPI**
- Урожайность кг/м²/нед

**Ключевая формула:**

```
Маржа/м²/нед = (Выручка_ARS/кг − COGS_ARS/кг) × урожай_кг/м²/нед
Цель: > 800 ARS/м²/нед (обновлять из реальных данных)
```

**Формат:**

```

Продукт      Rev/кг  COGS/кг  Марж/кг  кг/м²/нед  Марж/м²/нед
Огурцы:      [X]     [X]      [X]      [X]        [X]    [▲▼ vs цель]
```

### Модуль 4 — Cash Flow Monitor

Еженедельная позиция по денежным средствам.

```
💵 CASH FLOW — [период]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Открытие:         [X] ARS
+ Приход (продажи): [X] ARS
− Расход (поставщики, операции): [X] ARS
= Закрытие:       [X] ARS

Burn rate/нед:    [X] ARS
Runway:           [X] дней  [🟡 < 60 / 🔴 < 30]
USD эквивалент:   ~[X] USD (Blue)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[🔴 КРИТИЧНО: если runway < 30 дней]
```

**Пороги cash runway:**

- > 90 дней → ✅ нормально
- 31–90 дней → 🟡 мониторинг
- ≤ 30 дней → 🔴 КРИТИЧНО → автоматически создать blocked decision в "Задачи"

## Интеграции

- **`pepino-finance-tools`** — расчёты ARS/USD, NPV, unit economics формулы
- **`pepino-argentina-finance`** — FX-adjusted margin, рекомендации пересмотра цен
- **`pepino-weekly-review`** — контроллер поставляет данные для финансовой секции дайджеста
- **`pepino-shadow-ceo`** — при Critical alerts → автоматически создать blocked decision для CEO
- **`greenhouse-finance`** — для детальных финансовых расчётов

---

## Правила контроллера

- Никогда не использовать плановые данные как "факт" — только реальные записи из Sheets
- При нехватке данных: явно указать "данные не зафиксированы за период [X–Y]"
- ARS суммы всегда сопровождать USD эквивалентом (Blue Dollar)
- Исторический тренд: минимум 4 недели для trend analysis
- При Critical Alert → немедленно создать запись в "Алерты" с приоритетом 4-5
- Margin/м²/нед — ключевой KPI, отслеживать еженедельно
