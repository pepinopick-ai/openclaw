# 🎨 Pepino Pick — Бренд и Контент

Instagram контент, тексты для шефов, этикетки, WhatsApp скрипты, PR материалы.

---

## Голос бренда Pepino Pick

**Tone of voice:**

- Premium but approachable — не высокомерно, но статусно
- Farm-to-table authenticity — реальная ферма, реальные люди
- Научная точность + кулинарная страсть (EC/pH/PPFD → вкус на тарелке)
- Buenos Aires urban-farm вайб

**Ценности бренда:**

- Свежесть: harvest→delivery < 24h
- Уникальность: Tsitsak, Yerevan, shiitake, кимчи — этого нет у других
- Трассировка: каждый продукт с batch_id
- Устойчивость: waste-to-value, минимум отходов

**Ключевые продукты для контента:**
Tsitsak (перцы), Ереванский соус, шиитаке, вешенка, огуречное кимчи, лакто-огурцы, микрозелень, свежие огурцы

## 5 режимов

### Режим 1 — Instagram Post Generator

**3 поста в неделю:**

| День        | Формат                | Тема                          |
| ----------- | --------------------- | ----------------------------- |
| Понедельник | Producto de la Semana | Герой-продукт + история       |
| Среда       | Detrás de Escena      | Ферма / сбор / ферментация    |
| Пятница     | Chef Collab / Клиент  | Рецепт шефа или успех клиента |

**Структура поста:**

```
[Emoji] + Hook line (1 предложение — зацепить)
История продукта (2-3 предложения на испанском)
Сенсорное описание (запах, вкус, текстура)
Call to action
Хэштеги: #PepinoPick #FarmToTable #BuenosAires #[продукт] #[сезон]
```

**Шаблон Producto de la Semana:**

```
🌿 [ПРОДУКТ] de Pepino Pick

[История: откуда, как выращивается, что особенного — 2 предложения]

[Сенсорика: цвет, текстура, вкус, аромат — 1 предложение]

Disponible esta semana. Consultas por DM 📩

#PepinoPick #FarmToTable #BuenosAires #[продукт] #Gastronomia
```

**Шаблон Detrás de Escena:**

```
🏡 Así nace tu [продукт]

[Описание процесса — посев/инокуляция/ферментация — 2 предложения]

De nuestra granja a tu cocina en menos de 24 horas 🌱

#PepinoPick #FarmToKitchen #Sustentable #[продукт]
```

**Шаблон Chef Collab:**

```
👨‍🍳 Chef [имя] × Pepino Pick

"[Цитата шефа о продукте]"

[Продукт] de temporada disponible ahora.
¿Querés probarlo en tu restaurante? DM 📩

#PepinoPick #ChefCollab #Gastronomia #BuenosAires
```

### Режим 2 — Контент-план

**Месячный контент-план** (12-14 постов):

Структура по неделям:

- Неделя 1: Запуск сезонного продукта
- Неделя 2: Behind the scenes (ферма/производство)
- Неделя 3: Chef collab или client story
- Неделя 4: Ферментация / innovation / новинка

**Сезонные темы:**

- Осень: новый урожай грибов, ферментированные запасы
- Зима: согревающий Tsitsak, горячие соусы
- Весна: микрозелень, первые огурцы
- Лето: свежесть, кимчи как garnish

Вывод: конкретные темы для 12 постов месяца с продуктом, форматом, ключевым сообщением.

### Режим 3 — Коммерческие тексты

**WhatsApp outreach к шефу (первый контакт):**

```
Hola [nombre], soy Román de Pepino Pick 🌿
Cultivamos [продукт] de forma propia en [локация].
Tenemos algo que creo que va a interesarte para tu cocina.
¿Puedo mandarte una muestra esta semana para que lo pruebe?
Sin compromiso — solo queremos que lo conozca alguien que lo va a entender.
```

**Follow-up после дегустации (День 3):**

```
Hola [nombre] 👋
¿Cómo te fue con la muestra de [продукт]?
Si le encontrás aplicación, puedo armarte una propuesta a medida.
```

**Email для ресторанов (первый контакт):**

```
Asunto: [Producto] de cosecha propia para [Restaurante]

Hola [nombre],

En Pepino Pick cultivamos [продукт] en invernadero propio.
La diferencia: harvest-to-delivery en menos de 24 horas.

Productos disponibles esta temporada:
• [продукт 1]
• [продукт 2]

¿Podríamos dejarle una muestra esta semana?

Román | Pepino Pick
```

**Описание продукта для прайса:**

```
[ПРОДУКТ] — Pepino Pick
[2-3 предложения: происхождение, метод выращивания, вкусовой профиль]
Формат: [фасовка]. Срок: [дней] при [температура]°C.
```

### Режим 5 — PR и шеф-коллаборации

**Приглашение на дегустацию:**

```
Hola Chef [nombre] 👨‍🍳

En Pepino Pick cultivamos productos únicos para cocinas de autor:
[перечислить 2-3 продукта].

Tenemos [конкретный продукт] que creo que va a despertar tu creatividad.
¿Puedo dejarte una muestra esta semana — sin compromiso?

Solo queremos que lo pruebe alguien que lo va a entender.

Román | Pepino Pick 🌿
```

**Захват обратной связи шефа после дегустации:**

```
Продукт: [название]
Шеф: [имя] | Ресторан: [название] | Дата: [дата]

Оценка (1-5): [X]
Как использовал: [способ применения]
Комментарий вкуса: [цитата]
Интерес к закупке: да / нет / возможно
Запрос на другие продукты: [список]
```

**Идея co-branded контента:**
После того как шеф использует продукт → предложить пост:
"Chef [имя] de [ресторан] + nuestro [продукт]"
Отметить ресторан → взаимная аудитория.
