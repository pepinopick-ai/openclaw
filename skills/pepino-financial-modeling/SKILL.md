## Режим 1 — Сценарный анализ P&L

**Три сценария (base / bull / bear):**

```
СЦЕНАРИЙ P&L — [период]

| Метрика            | Bear     | Base     | Bull     |
|--------------------|----------|----------|----------|
| Выручка ARS/мес    | [X]      | [Y]      | [Z]      |
| Валовая маржа %    | [X]%     | [Y]%     | [Z]%     |
| EBITDA %           | [X]%     | [Y]%     | [Z]%     |
| Cash flow          | [X]      | [Y]      | [Z]      |
| Runway (мес)       | [N]      | [N]      | [N]      |

Допущения Bear: инфляция +10%/мес, продажи -20% vs план
Допущения Base: инфляция +5%/мес, продажи = план
Допущения Bull: новые клиенты +3/мес, product mix → premium
```

**Инфляционная корректировка (Аргентина):**

```
Реальный рост = (1 + Номинальный рост) / (1 + Инфляция) - 1
Реальная маржа ARS = Маржа_номинал / (1 + инфляция_период)
```

## Режим 3 — Break-even и ROI

**Break-even расчёт:**

```
BEP_единиц  = Постоянные_затраты / (Цена - Переменные_затраты_единицы)
BEP_выручка = Постоянные_затраты / (1 - Переменные / Выручка)

Пример:
  Постоянные затраты: 500,000 ARS/мес
  Цена вешенки: 3,500 ARS/кг
  Переменные на кг: 1,800 ARS
  BEP = 500,000 / (3,500 - 1,800) = 294 кг/мес
```

**ROI по продуктам:**

```
ROI АНАЛИЗ — [продукт]
  Инвестиции: [X] ARS (оборудование + субстрат + труд)
  Выручка за цикл: [X] ARS
  Себестоимость цикла: [X] ARS
  Прибыль цикла: [X] ARS
  Циклов в год: [N]
  Годовой ROI: [X]%
  Срок окупаемости: [N] мес
```

**DCF (Discounted Cash Flow):**

```
NPV = Σ(CF_t / (1+r)^t) - Инвестиции
r = ставка дисконтирования:
  ARS: 20–30%/год (реальная с учётом инфляции)
  USD: 8–12%/год
Terminal Value = FCF / (r - g)  где g = долгосрочный рост ~5%
```
