# pepino-chef-network

Управление сетью шефов и ресторанов Pepino Pick: реестр контактов, дегустации, сенсорные профили, Chef PR и коллаборации.

## Триггеры

шеф, шеф-повар, ресторан, дегустация, тастинг, сенсорный профиль, коллаборация, chef, вкус продукта, отзыв шефа, гастро, chef PR, упоминание в меню, рецепт шефа, приглашение шефа, новый шеф, база шефов, chef network

## Режимы работы

### 1. РЕЕСТР ШЕФОВ

**Структура карточки шефа:**

```
Имя: [ФИО]
Ресторан: [название]
Тип кухни: [итальянская / японская / fusion / аргентинская / etc]
Адрес: [район, город]
Instagram: [@handle]
Контакт: [телефон/email/WhatsApp]
Tier: [1 / 2 / 3]
Статус: [prospect / contacted / tasting_done / client / ambassador]
Потенциал ARS/мес: [сумма]
Последний контакт: [дата]
Следующий шаг: [действие]
Linked case: [CASE-YYYYMMDD-CHF]
```

**Tier классификация:**

- Tier 1: знаковые рестораны, высокий чек, влияние на рынок
- Tier 2: стабильные рестораны среднего сегмента, постоянный объём
- Tier 3: небольшие заведения, кафе, pop-up

**Команды:**

- `добавить шефа [имя] [ресторан]` → создать карточку
- `обновить статус [имя] → [новый статус]`
- `список шефов [Tier 1 / все / по статусу]`
- `следующие шаги по шефам` → кто требует follow-up

---

### 2. УПРАВЛЕНИЕ ДЕГУСТАЦИЯМИ

**Pipeline дегустации:**

```
prospect → invited → tasting_scheduled → tasting_done → follow_up → won / lost
```

**Шаблон подготовки к дегустации:**

```
Дата: [дата]
Шеф: [имя] / Ресторан: [название]
Продукты для дегустации:
  - [продукт 1] — [партия/дата сбора]
  - [продукт 2] — [партия/дата сбора]
Формат подачи: [сырой / приготовленный / с сопровождением]
Цель: [первый контакт / уточнение ассортимента / новый SKU]
Материалы: [каталог / образцы / КП]
Follow-up через: [N дней]
```

**После дегустации — обязательно фиксировать:**

- Что понравилось / не понравилось
- Какие продукты вызвали интерес
- Объём и частота — какие потребности
- Ценовой диапазон, который принял
- Следующий шаг (заказ / повторная дегустация / отказ)

---

### 3. СЕНСОРНЫЕ ПРОФИЛИ ПРОДУКТОВ

Накапливаем обратную связь от шефов для улучшения продукта и маркетинга.

**Формат сенсорного профиля:**

```
Продукт: [название]
Партия: [BATCH-ID]

Отзывы шефов:
  Шеф [имя] ([ресторан]):
    Вкус: [описание]
    Текстура: [описание]
    Аромат: [описание]
    Применение: [как использует в меню]
    Оценка: [1-10]
    Цитата: "[прямая речь]"

Сводный профиль:
  Сильные стороны: [что нравится большинству]
  Зоны улучшения: [что требует доработки]
  Лучшее применение: [топ-3 способа использования]
  Ценовое восприятие: [дорого / справедливо / доступно]
```

**Продукты для профилирования:**

- Вешенка свежая
- Шиитаке свежий
- Ogurec Pepino (огурцы)
- Ферментированный Tsitsak
- Кимчи / квашеные продукты
- Экспериментальные SKU (новые)

---

### 4. CHEF PR И КОЛЛАБОРАЦИИ

**Типы коллабораций:**

- **Меню-упоминание**: продукт фигурирует в меню ресторана
- **Авторский рецепт**: шеф разрабатывает рецепт специально под наш продукт
- **Гастро-вечер**: совместное мероприятие (ужин, pop-up, мастер-класс)
- **Контент-коллаборация**: шеф снимает контент с нашим продуктом для Instagram
- **Амбассадор**: долгосрочное партнёрство, публичное позиционирование

**Шаблон предложения коллаборации:**

```
Шеф [имя], ресторан [название]
Формат: [тип коллаборации]
Наше предложение: [что даём — продукты, упоминание, оплата, etc]
Их вклад: [что ожидаем — контент, рецепт, мероприятие]
Срок: [период]
KPI: [охват / заказы / контент]
Согласовано: [да / нет / в переговорах]
```

**Трекинг активных коллабораций:**

```
[Шеф] — [ресторан] — [тип] — [статус] — [дедлайн] — [результат]
```

### 5. PIPELINE КОНВЕРСИИ

**Воронка шеф → клиент:**

```
Всего шефов: [N]
├─ Prospect:          [N] — ещё не контактировали
├─ Contacted:         [N] — вышли на контакт
├─ Tasting Done:      [N] — прошли дегустацию
├─ Follow-Up:         [N] — ждём решения
├─ Won (клиент):      [N] — постоянные клиенты
├─ Lost:              [N] — отказали
└─ Ambassador:        [N] — партнёры-амбассадоры
```

**Конверсионные метрики:**

- Contact rate: contacted / prospect
- Tasting rate: tasting_done / contacted
- Win rate: won / tasting_done
- Среднее время от первого контакта до первого заказа

**Еженедельный отчёт (каждый понедельник):**

```
🍽️ CHEF NETWORK — [дата]

Новые контакты на неделе: [N]
Дегустации: [N]
Новые клиенты: [N]
Потерянные: [N]

Требуют follow-up (>7 дней без контакта):
  - [Шеф] — [ресторан] — последний контакт [дата]
  - ...

Дегустации запланированы:
  - [Шеф] — [дата]

Активные коллаборации:
  - [Шеф] — [тип] — [статус]
```

## Интеграция

| Навык                   | Когда                                              |
| ----------------------- | -------------------------------------------------- |
| `pepino-sales-crm`      | Шеф → Tier 1/2 клиент, pipeline синхронизация      |
| `pepino-brand`          | Коллаборация → контент для Instagram, пост с шефом |
| `pepino-qa-food-safety` | Дегустация → batch ID для трассировки              |
| `greenhouse-sheets`     | Запись дегустации, обновление статуса шефа         |
| `pepino-knowledge`      | Сенсорный профиль → Obsidian 11_people/clients/    |
| `pepino-innovation-lab` | Отзыв шефа → новый SKU или улучшение продукта      |

## Формат ответа

```
case_id: CASE-[YYYYMMDD]-CHF
intent: chef_[registry|tasting|sensory|pr|pipeline]

[результат по запрошенному режиму]

linked_agents: [pepino-sales-crm, pepino-brand, etc]
next_actions:
  - [конкретное следующее действие]
```

## Правила

- Никогда не звонить шефу без предварительного согласования времени
- После каждой дегустации — follow-up в течение 48 часов
- Сенсорный профиль обновляется после каждой дегустации с новым шефом
- Коллаборации уровня Ambassador требуют апрува CEO
- Все договорённости с шефами фиксировать в письменном виде (WhatsApp минимум)
