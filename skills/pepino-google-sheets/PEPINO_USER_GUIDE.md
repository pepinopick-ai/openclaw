## Что это такое

Pepino Pick Agent OS — это твоя **команда цифровых помощников**, которая работает 24/7.
71 скрипт автоматизации, 42 задачи по расписанию, 6 n8n workflow'ов, Digital Twin фермы.
Ты общаешься с ними через Telegram или терминал — как с реальными сотрудниками.

## Часть 2. Каждый день — 10 минут

### Утро (5 минут)

**Что приходит автоматически в Telegram к 07:00:**

- Утренний бриф (погода, алерты, план)
- Чеклист дня (что собрать, кому доставить, что починить)
- Маршрут доставки (если есть заказы)
- Состояние склада (что заканчивается)

**Что делать:** прочитать и отреагировать на красные алерты.

**Или в терминале:**

```
pepino twin
```

Покажет всю ферму одним экраном:

```
--- INVENTORY ---
Products: 3 | RED:3 YEL:0 GRN:0
  !!! Огурец: 0kg (0d left)

--- FINANCIAL ---
Week: 1.5M ARS rev | Margin 96%
Blue: 1405/1425

--- CLIENTS ---
Active:4 AtRisk:1 Churned:11
Expected orders this week: 5

--- 2-WEEK FORECAST ---
Revenue: ~2.1M ARS | Margin: ~87%
```

### В течение дня

**Записывай расходы** — это самое важное для точной маржи:

```
pepino expense "субстрат 5000"
pepino expense "доставка 3500 наличные"
pepino expense "зарплата Мигель 45000"
pepino expense "электричество 12000 перевод"
```

Система автоматически определяет категорию и записывает в Sheets.

**Посмотреть расходы:**

```
pepino expense --list    → за сегодня
pepino expense --week    → за неделю с разбивкой
```

Пример:

```
Персонал: 45 000 ARS (69%)
Коммунальные: 12 000 ARS (18%)
Материалы: 5 000 ARS (8%)
ИТОГО: 65 500 ARS | Среднедневной: 9 357 ARS
```

### Вечер (2 минуты)

```
pepino sales today     → что продали
pepino pnl week        → P&L за неделю
```

## Часть 4. Что работает само (42 задачи по расписанию)

### Утренний pipeline (06:00)

Автоматическая цепочка из 5 шагов:

1. Оптимизация маршрута доставки
2. Операционный чеклист дня
3. Проверка остатков склада
4. Пересчёт агрегатов P&L
5. Утренний бриф → Telegram

### Вечерний pipeline (20:30)

1. Пересчёт агрегатов
2. Daily P&L и маржа
3. Проверка полноты данных
4. Отчёт о расходах на AI
5. Агрегация алертов

### Воскресный pipeline (17:00)

1. Анализ потерь производства
2. План производства на неделю
3. Извлечение бизнес-знаний
4. CEO-дайджест → Telegram

## Часть 5. Digital Twin — модель фермы

Это виртуальная копия твоей фермы, которая показывает всё одним взглядом.

```
pepino twin
```

7 секций:

1. **Производство** — что растёт, на какой стадии, когда собирать
2. **Склад** — что есть, на сколько хватит (🟢🟡🔴)
3. **Финансы** — выручка, расходы, маржа, тренд, курс Blue
4. **Клиенты** — активные, под угрозой, ушедшие
5. **Погода** — температура, влажность, влияние на растения
6. **Нагрузка** — открытые задачи, просроченные, алерты
7. **Прогноз 2 недели** — сколько заработаем, потратим, соберём

**HTML-версия** (красивый дашборд):

```
pepino twin html
→ открой /tmp/pepino-twin.html в браузере
```

## Часть 10. Защита от спама в Telegram

Система отправляет десятки сообщений в день. Чтобы не засорять чат:

- **Одинаковые сообщения** не повторяются 4 часа
- **Максимум 5 сообщений** в час на один тред
- **Ночью (22:00-06:00)** только критические проходят, остальные копятся
- **"CRITICAL" и "P1"** всегда доставляются мгновенно

## Часть 12. Диагностика проблем

| Проблема                | Что делать                                 |
| ----------------------- | ------------------------------------------ |
| "Дашборд пустой"        | `pepino status` → проверить сервисы        |
| "Бриф не пришёл"        | `pepino logs morning-brief`                |
| "Данные не обновляются" | Пересчёт в 07:00 и 20:30 автоматически     |
| "Маржа 100%"            | Расходы не внесены! `pepino expense "..."` |
| "Контейнер упал"        | Healthcheck починит через 30 мин           |
| "Что-то сломалось"      | `pepino test` → покажет что именно         |
| "Telegram спамит"       | Throttle ограничивает до 5/час             |
